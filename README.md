# SGAI_2023

Dependencies:
- PIL
  - `pip3 install pillow`
- TKinter
  - `pip3 install tk`
- Pytorch 
  - `pip3 install pytorch`

To run, type "python3 main.py" in the terminal.


Alternatively:
Set up new Conda environment with:
`conda create -n BWSI_2024 python=3.9 pillow tk pytorch pyyaml torchvision`
